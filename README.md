# musiconreplit
在 Replit 部署 YesPlayMusic 网页端（自带NeteaseCloudMusicApi）

先运行 first.sh 初始化
在shell中运行 `sh first.sh` 进行初始化，初始化完成后点绿色Run 运行

教育版：

将以下代码粘贴至Replit Shell后回车

`git clone https://github.com/valetzx/musiconreplit && mv -b musiconreplit/* ./ && mv -b musiconreplit/.[^.]* ./ && rm -rf *~ && rm -rf musiconreplit`

当加载完 Loading Nix environment... 后点击绿色 ▶ Run
